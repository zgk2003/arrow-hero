About this game

Embedded Arrows is a simple but fun reaction game where the goal is to match the arrows that show up on the screen using the buttons on your board.

This means that if you have more than one consecutive you only have to press once to validate the following arrows of the same direction.

Key Guide

UP: left + right button
LEFT: left button
RIGHT: right button
PAUSE/UNPAUSE: touch slider
RESTART: touch slider
